🎯 Core Concepts

🔹 Algorithm

Finite, ordered sequence of unambiguous steps solving a problem.

🔹 Two Phases

  • Problem-solving → create algorithm
  • Implementation → code in programming language

🔹 Representations

  • Pseudocode: informal English-like
  • Flowchart: graphical (ovals, rectangles, diamonds)

🔹 Control Structures

  • Sequence → step-by-step
  • Decision (if-then-else)
  • Loop (while, for)
  • Case (multi-selection)
Flowchart symbols Oval: Start/Stop  |  Rectangle: Process  |  Parallelogram: I/O  |  Diamond: Decision
🧠 Quadratic Equation & USEP Grade Example

Discriminant (D = b²-4ac) → D>0: real & unequal; D=0: real & equal; D<0: imaginary.
USEP Grade: 97-100% = 1.0 (Passed), 75% = 3.0, below 75 = 5.0 Failed.
